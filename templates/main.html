Digit Recognizer

Github

Single digit recognizer made with Flask and the Python Keras library trained on the MNIST dataset using a CNN.

Tips for best recognition:

  • Write the digit as large as possible
  • Keep the digit as centered as possible
  • Avoid drawing anything but the single digit
  • Recognition may take around 5 seconds

Known limitations:

  • Cannot recognize multiple digits at a time
  • Sideways or upside down digits are unlikely to be recognized
